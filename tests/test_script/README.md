# Скрипты FSRS плагина

Этот каталог содержит вспомогательные скрипты для разработки и тестирования плагина FSRS для Obsidian.

## `rebuild-log.sh` - Скрипт пересборки и сбора логов

Скрипт автоматически пересобирает плагин (`npm run build`), делает паузу для обновления Obsidian (по умолчанию 3 секунды) и собирает файлы и логи из тестового хранилища.

### Настройка среды

Пользователь предворительно должен 
- открыть хранилище,
- активировать логирующий плагин (Logstravaganza)
- открыть вкладку с нужным контентом (нужно с код-блоком плагина)

### Запуск скрипта

```bash
cd /path/to/FSRS-plugin_Task_3
./scripts/rebuild-log.sh
```

Если скрипт не имеет прав на выполнение:
```bash
chmod +x scripts/rebuild-log.sh
./scripts/rebuild-log.sh
```

### Получение справки

Для просмотра всех параметров и справки используйте:
```bash
./scripts/rebuild-log.sh --help
```

### Пример с пользовательским хранилищем

```bash
./scripts/rebuild-log.sh --vault-path /home/user/my-vault --delay 5
```
