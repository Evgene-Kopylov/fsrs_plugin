# Использование блока fsrs-now в FSRS плагине для Obsidian

## 📖 Введение

Блок `fsrs-now` — это автоматически формируемый список карточек FSRS, которые готовы к повторению. Вместо неудобных уведомлений, которые нельзя использовать для навигации, теперь все карточки отображаются в виде форматированного блока кода в любом markdown-файле вашего хранилища.

## 🚀 Как использовать

### 1. Добавить блок fsrs-now в текущий файл

Откройте **командную палитру** (Ctrl+P или Cmd+P) и выполните команду:
```
Найти карточки для повторения
```

Плагин:
1. Просканирует всё ваше хранилище на наличие карточек FSRS
2. Определит, какие карточки просрочены (дата повторения наступила)
3. Вставит или обновит блок `fsrs-now` в текущем активном файле

### 2. Где может находиться блок

Вы можете добавить блок `fsrs-now` в любой markdown-файл:
- В файл с вашими ежедневными заметками
- В специальный файл для повторения (например, `FSRS Review.md`)
- В файл со статистикой обучения
- Куда угодно, где вам удобно видеть список для повторения

## 📊 Формат вывода

Блок `fsrs-now` имеет следующий формат:

````fsrs-now
# Карточки для повторения (5)

*Обновлено: 15.01.2024, 14:30:23*

## 1. [[notes/biology/cell-structure.md]]
- **Просрочено**: 2 дня 3 часа
- **Состояние**: Review | **Повторений**: 3 | **Ошибок**: 0
- **Стабильность**: 5.21 | **Сложность**: 0.45
- **Дата повторения**: 13.01.2024, 11:00:00
- **Действия**: [[notes/biology/cell-structure.md|Открыть]] | [Повторить](obsidian://fsrs-review?file=notes/biology/cell-structure.md)

## 2. [[notes/math/calculus-derivatives.md]]
- **Просрочено**: 5 часов
- **Состояние**: Learning | **Повторений**: 1 | **Ошибок**: 0
- **Стабильность**: 1.50 | **Сложность**: 0.75
- **Дата повторения**: 15.01.2024, 09:00:00
- **Действия**: [[notes/math/calculus-derivatives.md|Открыть]] | [Повторить](obsidian://fsrs-review?file=notes/math/calculus-derivatives.md)

--- 
*Показано: 2 из 5 карточек (3 скрыто)*
---
*Для обновления списка выполните команду "Найти карточки для повторения"*
````

## 🧭 Навигация по карточкам

### Кликабельные ссылки

Каждая карточка в списке содержит:

1. **Ссылка на файл**: `[[notes/biology/cell-structure.md]]` — кликабельная ссылка для перехода к файлу
2. **Ссылка "Открыть"**: Открывает карточку в новой вкладке Obsidian
3. **Ссылка "Повторить"**: Специальная ссылка для быстрого перехода к повторению карточки (функционал в разработке)

### Сортировка карточек

Карточки сортируются по дате повторения:
- **Самые старые** первыми (самая большая просрочка)
- **Самые новые** в конце (минимальная просрочка)

### Лимит отображения

По умолчанию отображается **первые 30 карточек**. Если карточек больше:
- Показаны первые 30
- В конце блока указано, сколько карточек скрыто
- Пример: `*Показано: 30 из 45 карточек (15 скрыто)*`

## 🔄 Обновление списка

### Автоматическое обновление

При повторном выполнении команды "Найти карточки для повторения":
1. Существующий блок `fsrs-now` будет **заменён** на новый
2. Все изменения в блоке будут обновлены
3. Если в файле несколько блоков `fsrs-now` — будет обновлён только первый, остальные удалены

### Ручное обновление

Просто выполните команду повторно, когда хотите обновить список:
- После завершения сессии повторений
- В начале дня для планирования
- После добавления новых карточек FSRS

## 🎯 Практические сценарии использования

### Сценарий 1: Ежедневное повторение

1. **Утром**: Выполните команду "Найти карточки для повторения" в файле `Daily/2024-01-15.md`
2. **Просмотрите список**: Определите, какие карточки нужно повторить сегодня
3. **Выполните повторения**: Кликайте на ссылки и повторяйте карточки
4. **После завершения**: Обновите список командой

### Сценарий 2: Фокус на определённой теме

1. **Создайте файл**: `Topics/Biology Review.md`
2. **Добавьте блок**: Выполните команду в этом файле
3. **Используйте как панель управления**: Все карточки по биологии будут в одном месте
4. **Организуйте по папкам**: Если хотите видеть карточки только из определённых папок

### Сценарий 3: Мониторинг прогресса

1. **Создайте файл статистики**: `FSRS Statistics.md`
2. **Добавьте блок в начале файла**: 
   ```markdown
   # Статистика повторений
   *Обновлено ежедневно*
   
   ```fsrs-now
   # Карточки для повторения (15)
   ...
   ```
   ```
3. **Добавьте свою статистику** под блоком
4. **Наблюдайте за прогрессом**: Как меняется количество карточек со временем

## 💡 Советы и рекомендации

### 1. **Используйте в ежедневных заметках**
Добавляйте блок в ваш daily note, чтобы видеть, что нужно повторить сегодня.

### 2. **Фильтрация через поиск**
Хотите видеть только карточки из определённой папки? Используйте Obsidian поиск:
- Откройте файл с блоком `fsrs-now`
- Используйте поиск в файле (Ctrl+F)
- Ищите по пути папки, например: `notes/biology/`

### 3. **Копирование ссылок для планирования**
Можете копировать ссылки на карточки и добавлять их в свой планировщик задач.

### 4. **Цветовое кодирование**
Используйте CSS сниппеты Obsidian для цветового кодирования состояния карточек:
- `Review` — зелёный
- `Learning` — синий  
- `Relearning` — оранжевый
- `New` — серый

### 5. **Интеграция с другими плагинами**
- **Dataview**: Можно экспортировать данные из блока для анализа
- **Tasks**: Можно создавать задачи на основе карточек для повторения
- **Calendar**: Планируйте повторения в календаре

### Обработка нескольких блоков
Если в файле есть несколько блоков `fsrs-now`:
- **Обновляется** только первый найденный блок
- **Остальные блоки удаляются** (чтобы избежать дублирования)

### Производительность
- **Сканирование всех файлов**: Может занять несколько секунд для больших хранилищ
- **Кэширование**: Не реализовано (планируется в будущих версиях)
- **Обновление**: Только при явном выполнении команды

## ❓ Часто задаваемые вопросы

### Q: Почему не отображаются все карточки?
**A**: Отображаются только первые 30 карточек для удобства просмотра. Если у вас больше карточек, они будут скрыты.

### Q: Можно ли изменить лимит отображения?
**A**: В текущей версии лимит фиксированный (30 карточек). В будущих версиях планируется добавить настройки.

### Q: Как удалить блок `fsrs-now`?
**A**: Просто удалите блок кода из файла вручную или оставьте пустым и выполните команду обновления.

### Q: Работает ли на мобильных устройствах?
**A**: Да, блок `fsrs-now` работает на всех платформах, где работает Obsidian.

### Q: Можно ли экспортировать данные в CSV?
**A**: В текущей версии нет, но можно копировать текст из блока и конвертировать вручную.
