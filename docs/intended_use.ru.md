# Руководство по использованию FSRS

- [🇷🇺](intended_use.ru.md) <
- [🇺🇸](intended_use.en.md)

Этот гайд покажет, как начать использовать плагин после установки.
Всё работает прямо в Obsidian — никакие внешние сервисы не нужны.

---

## 0. Установка

Плагин пока отсутствует в каталоге сообщества Obsidian —
установите его через **BRAT** (Beta Reviewers Auto-update Tester):

1. Установите [BRAT](https://github.com/TfTHacker/obsidian42-brat)
   из **Settings → Community plugins → Browse**
2. Откройте **Settings → BRAT → Add Beta plugin**
3. Введите URL репозитория: `https://github.com/Evgene-Kopylov/fsrs_plugin`
4. Включите плагин в **Settings → Community plugins**

BRAT будет автоматически отслеживать обновления.

## 1. Инициализировать карточку

После установки и включения плагина откройте заметку которую планируете повторять.

Вызовите палитру команд (`Ctrl/Cmd+P`) и выполните:

### FSRS: Добавить поля FSRS в шапку файла

![команда «Добавить поля FSRS» в палитре](images/add-fsrs-fields-command.png)

Плагин добавит в frontmatter заметки пустой массив `reviews: []`
для регистрации повторений, а сразу после шапки — кнопку повторения:

````markdown
---
reviews: []
---

```fsrs-review-button
```
````

В режиме чтения code-блок превратится в кнопку.
При клике открывается вьюшка с вариантами оценки:
**Again**, **Hard**, **Good**, **Easy**.

![кнопка повторения в режиме просмотра](images/btn-in-preview.png)

Кнопка кликабельна и в предпросмотре при наведении из таблицы —
это основной способ повторения (см. раздел 2).

Также повторять можно командой **FSRS: ✓ Повторить текущую карточку** —
без кнопки, прямо из палитры. А добавлять или не добавлять кнопку —
можно определить в настройках.

В массив `reviews` плагин записывает даты и оценки повторений.
После пары повторений frontmatter будет выглядеть примерно так:

```yaml
---
reviews:
  - date: "2025-03-15T12:00:00Z"
    rating: 2
  - date: "2025-03-17T08:00:00Z"
    rating: 3
---
```

Значения `rating`:
**0** = Again (снова),
**1** = Hard (сложно),
**2** = Good (хорошо),
**3** = Easy (легко).

Если авто-вставка кнопки отключена в настройках —
вставьте её вручную командой **FSRS: Вставить кнопку повторения**.

## 2. Создать таблицу карточек

Теперь откройте заметку, где хотите видеть выборку по своим карточкам,
и выполните команду

### FSRS: Вставить таблицу fsrs-table по умолчанию

![команда «Вставить таблицу fsrs-table» в палитре](images/insert-table.png)

Она вставит готовый блок `fsrs-table` с SQL-подобным запросом:

````markdown
```fsrs-table
SELECT file as "Карточка",
       retrievability as "R",
       stability as "S",
       difficulty as "D",
       due as "Следующее"
LIMIT 20
```
````

В режиме просмотра блок отрендерится в таблицу с карточками,
отсортированными по умолчанию (по срочности — самые забытые сверху).

![отрендеренная fsrs-table с карточками](images/fsrs-table-demonstration.png)

### Что значат колонки

FSRS рассчитывает три параметра для каждой карточки
на основе истории повторений (дата и оценка):

| Поле | Смысл | Как меняется |
| --- | --- | --- |
| **Difficulty** (D) | Насколько материал сложный | Почти не меняется — сложная тема остаётся сложной |
| **Stability** (S) | Насколько прочно воспоминание, в днях | Растёт с каждым успешным повторением |
| **Retrievability** (R) | Вероятность вспомнить прямо сейчас | Падает каждую секунду после повторения |

Когда Retrievability опускается ниже порога — карточка появляется в списке на повторение.
Порог настраивается: хотите 90% сохранения — повторяете чаще, достаточно 80% — реже.

Полный список полей — в [README](../README.ru.md###доступные-поля).
Примеры настройки запросов — в [sql-syntax.md](sql-syntax.md).

## 3. Повторять без перехода к заметке

Это основной сценарий использования плагина.
Наведите курсор на имя файла в таблице.

Появится всплывающее окно с содержимым заметки,
а внутри — та самая кнопка повторения, кликабельная из предпросмотра.

![всплывающее окно с контентом и кнопкой повторения](images/demo-tanzanit.gif)

Цикл повторения остаётся в одном окне:

1. Открываете заметку с таблицей (например, ежедневную заметку).
2. Таблица показывает все карточки — самые срочные сверху.
3. Наводите на карточку — всплывает содержимое.
4. Читаете, нажимаете оценку — карточка обновляется.
5. Переходите к следующей строке.

Таким образом можно пролистать все карточки за пару минут,
ни разу не переключившись в другую заметку.

  для первой заметки (кнопка добавится автоматически)
- [ ] Выполнена команда **FSRS: Вставить таблицу fsrs-table по умолчанию**
- [ ] Всё готово к повторению
