# Задача 4: Версия в README.ru.md

## Описание

В нижнем колонтитуле `README.ru.md` указана устаревшая версия `0.4.5`. Нужно обновить до `0.4.6`.

## Где править

`FSRS-plugin/README.ru.md`

## Что исправить

```
*Версия плагина: 0.4.5*
```

→

```
*Версия плагина: 0.4.6*
```

## Критерий готовности

- `grep "Версия плагина:" README.ru.md` показывает `0.4.6`
