# Задача 6: docs/intended_use.ru.md — версия и ссылка

## Описание

Исправить устаревшую версию плагина и неверную ссылку language switcher
в файле `docs/intended_use.ru.md`.

## Что не так

1. **Версия**: внизу страницы указано `Plugin version: 0.4.5`, нужно `0.4.6`.
2. **Language switcher**: ссылка `[🇺🇸](README.md)` ведёт на README,
   а должна вести на английскую версию этого же документа — `[🇺🇸](intended_use.md)`.

## Где находится

`docs/intended_use.ru.md`, последняя строка:

```
*Plugin version: 0.4.5*
```

И первая строка после `#`-заголовка:

```
- [🇺🇸](README.md)
```

## Что сделать

1. `*Plugin version: 0.4.5*` → `*Plugin version: 0.4.6*`
2. `[🇺🇸](README.md)` → `[🇺🇸](intended_use.md)`
   (ссылка должна указывать на английскую версию руководства,
   а не на главный README)
