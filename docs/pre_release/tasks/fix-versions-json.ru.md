# Задача 2: дополнить versions.json

## Описание

Файл `versions.json` содержит маппинг версий плагина на минимальную версию Obsidian.
Сейчас там только одна запись: `"0.1.1": "0.1.0"`. Для релиза v0.4.6 нужно добавить актуальную запись.

## Текущее состояние

```json
{
	"0.1.1": "0.1.0"
}
```

## Целевое состояние

```json
{
	"0.1.1": "0.1.0",
	"0.4.6": "0.15.0"
}
```

## Шаги

1. Открыть `versions.json`
2. Добавить строку `"0.4.6": "0.15.0"` (если `minAppVersion` в `manifest.json` будет изменена на `0.15.0`)
3. Проверить форматирование JSON

## Критерии готовности

- `versions.json` содержит запись для версии `0.4.6`
- Значение `minAppVersion` совпадает с `manifest.json`
- Файл является валидным JSON
