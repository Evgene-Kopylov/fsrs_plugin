# FSRS 术语表

## reps（复习次数）

卡片的所有复习次数。每次点击（重来、困难、良好、简单）+1。
即使「重来」也算一次复习 — 算法获得了反馈。

## lapses（失误次数）

成功复习后再次点击「重来」的计数。
新卡片第一次「重来」不算失误。
影响稳定性和难度。

## 为什么「重来」算一次复习？

「重来」是一次暴露出薄弱结果的复习，而非产生薄弱结果。

## DSR 模型

**DSR**（难度 — 稳定性 — 可提取性）— 基础三元组参数，
描述 FSRS 记忆模型中卡片的状态。
顺序 **D → S → R** 反映了模型的逻辑顺序：

1. **D**（Difficulty / 难度）— 材料的静态特征，
   反映其记忆难度。不随时间变化。
   范围：从 0（非常容易）到 10（非常困难）。

2. **S**（Stability / 稳定性）— 记忆牢固程度，以天为单位。
   - 表示多少天后回忆概率将降至
     90%（可配置参数）。
   - 成功复习时增长，失误（「重来」）时下降。

3. **R**（Retrievability / 可提取性）— 动态概率（0.0–1.0）
   表示当前时刻能回忆起来的概率。
   - 根据 S 和距上次复习的时间按公式计算：
     时间越久，R 越低。
   - 成功回答后 R = 1.0（100%），然后指数下降。
   - 在历史表格中显示为百分比形式的
     `可提取性 (R)`（回答前的值）。

逻辑顺序：从材料的基本特征（D）到掌握牢固程度（S），再到当前回忆概率（R）。
