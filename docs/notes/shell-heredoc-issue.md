# Heredoc не работает в /bin/sh

## Проблема

При попытке использовать heredoc в shell-командах через терминал:

```sh
python3 << 'EOF'
...
EOF
```

Возникает ошибка:

```
/bin/sh: 16: Syntax error: end of file unexpected (expecting ")")
```

## Причина

Оболочка `/bin/sh` (dash на Ubuntu) не поддерживает многострочные heredoc
в некоторых контекстах, либо инструмент терминала некорректно их передаёт.

## Решение

### python3 -c с переносами строк
```sh
python3 -c "строка1\nстрока2\n..."
```

### Временный .py файл
Для объёмных скриптов — записать в файл через edit_file, выполнить, удалить.
