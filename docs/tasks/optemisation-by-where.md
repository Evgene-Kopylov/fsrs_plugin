## Ситуация

При большом количестве карточек, тысячах, одна их обработка из кеша 
становится ощутимо накладной.

## Идея

Держать в кеше только саму непосредственно выборку по условию, если задано.


> Подход 4 (более простой):** Если у рендерера есть WHERE параметры, то при первом рендере, после того как WASM отфильтровал карточки, сохранять в `this.cachedCards` только отфильтрованные, а не все. Тогда при сортировке (когда `lastAction === "sort"`) используется уже отфильтрованный подмассив.
