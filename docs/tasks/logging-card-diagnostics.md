# Логирование диагностики карточки

Добавить в лог достаточно информации для самоконтроля:

- Сколько карточек всего (totalCount)
- Сколько показывается (лимит демонстрации)
- Какие фильтры (WHERE) применены
- Какой ORDER BY и LIMIT задан
- Сколько карточек в текущей выборке

Лог-строка должна быть одной информативной строкой, чтобы пользователь мог быстро понять, что произошло с выборкой.
