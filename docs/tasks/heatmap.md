# Тепловая карта (Heatmap)

Компонент мотивации — показывает сколько повторений сделано по дням.

## Идея

Классический виджет «activity heatmap», как на GitHub: сетка дней, цвет зависит от количества повторений в этот день.

## Требования

- Отображение в виде отдельного блока `fsrs-heatmap` или вкладки
- Каждый день — ячейка, цвет = интенсивность (0 повторений → нет закраски, много → яркий цвет)
- Данные из WASM-кэша (даты повторений по всем карточкам)
- Поддержка локализации (месяцы, дни недели)
