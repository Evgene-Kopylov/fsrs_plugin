
В папке `docs/tasks/` можно найти задачи. Одна задача - один файл. 

В `docs/tasks/index.md` перечень задачь с - [ ] | - [x] чекбоксами выполнения. 

Порядок работ над задачами
- отметить свободную как в работе [>]
- выполнить, провести тесты и сделать коммит
- коммит принимает пользователь, проводит ручные тесты если надо, делает замечаня. 
- довести задачу до принятия пользователем.
- отметить задачу выполненной и взять соледующую свободную
- если задач в `index.md` нет, следует обновить его задачами из папки `docs/tasks/`
