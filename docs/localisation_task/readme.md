### Как использовать

1. Открывайте файл задачи (например, `1.md`).
2. Выполняйте описанные в нём действия.
3. После завершения поставьте `[x]` в чекбокс напротив задачи (в этом индексном файле).
4. Делай коммит.

> **Примечание:** Задачи можно выполнять последовательно, так как каждая опирается на результаты предыдущих (например, локализация статус-бара использует уже созданную систему `i18n`).

---

## Общая структура локализации

- `src/locales/en.json` – английские строки  
- `src/locales/ru.json` – русские строки  
- `src/utils/i18n.ts` – менеджер локализации (функции `t()`, `setLocale()`, `showNotice()`)

После выполнения всех задач плагин будет полностью локализован, а пользователь сможет выбирать язык в настройках.
