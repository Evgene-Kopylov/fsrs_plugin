## Задание для агента №3: Локализация статус-бара

### Цель
Перевести все текстовые элементы статус-бара плагина (нижняя строка состояния Obsidian) с использованием созданной системы локализации. Статус-бар отображает информацию о текущей карточке: готовность к повторению, время до следующего повторения, сообщения об ошибках и т.д. Также перевести контекстное меню статус-бара (правый клик).

### Контекст
Файл `src/ui/status-bar-manager.ts` содержит жёстко зашитые русские строки в нескольких местах:
- Текст статус-бара (например, `FSRS: due!`, `FSRS: loading...`, `FSRS: not FSRS`, `FSRS: error`).
- Тултип (всплывающая подсказка при наведении).
- Пункты контекстного меню (правый клик): "Добавить поля FSRS в шапку файла", "Показать историю повторений", "Показать справку по синтаксису fsrs-table".
- Сообщения о времени до досрочного повторения (например, `FSRS: Wait 40 minutes`).

Необходимо заменить все эти строки на вызовы `i18n.t()` и добавить соответствующие ключи в файлы локализации.

#### 2. Модифицировать `status-bar-manager.ts`

**Импорт:**
```typescript
import { i18n, showNotice } from '../utils/i18n';
```

**В методе `init()`:**
- Заменить текст при создании:  
  `this.textSpan.textContent = " FSRS: loading...";`  
  → `this.textSpan.textContent = " FSRS: " + i18n.t('statusBar.loading');`
- Установить тултип:  
  `this.statusBarItem.title = i18n.t('statusBar.tooltip.default');`

**В методе `updateStatusBar()`:**
- Заменить все `textContent` и `title` в соответствии с новыми ключами.

Примеры замен:

**Нет активного файла:**
```typescript
if (this.textSpan) this.textSpan.textContent = " FSRS: " + i18n.t('statusBar.no_file');
this.statusBarItem.title = i18n.t('statusBar.tooltip.no_file');
```

**Не FSRS карточка:**
```typescript
if (this.textSpan) this.textSpan.textContent = " FSRS: " + i18n.t('statusBar.not_fsrs');
this.statusBarItem.title = i18n.t('statusBar.tooltip.not_fsrs');
```

**Готова к повторению:**
```typescript
if (this.textSpan) this.textSpan.textContent = " FSRS: " + i18n.t('statusBar.due');
this.statusBarItem.title = i18n.t('statusBar.tooltip.due');
```

**Ошибка:**
```typescript
if (this.textSpan) this.textSpan.textContent = " FSRS: " + i18n.t('statusBar.error');
this.statusBarItem.title = i18n.t('statusBar.tooltip.error');
```

**Карточка не просрочена, ожидание:**
```typescript
if (minutesSinceLastReview < minInterval) {
  const remainingMinutes = minInterval - minutesSinceLastReview;
  const noun = getRussianNoun(remainingMinutes, "минуту", "минуты", "минут");
  if (this.textSpan) this.textSpan.textContent = ` FSRS: ${i18n.t('statusBar.wait_early', { minutes: remainingMinutes, noun })}`;
  this.statusBarItem.title = i18n.t('statusBar.tooltip.early_available', { minutes: remainingMinutes, noun, date: formattedDate });
} else {
  if (this.textSpan) this.textSpan.textContent = ` FSRS: ${i18n.t('statusBar.next_review', { date: formattedDate })}`;
  this.statusBarItem.title = i18n.t('statusBar.tooltip.next_scheduled', { date: formattedDate });
}
```

#### 3. Удалить старые русские строки
- Убедиться, что в `status-bar-manager.ts` больше нет жёстко зашитых русских текстов (кроме вызовов `getRussianNoun`, который остаётся как вспомогательная функция для склонения – он не локализуется, т.к. числа и склонения специфичны для языка, но для английского эта функция не нужна, т.к. там нет склонений. Однако для упрощения оставляем как есть, позже можно будет переработать с использованием `Intl`).

### Критерии выполнения
- Все тексты в статус-баре (основной текст, тултип, пункты меню) заменены на вызовы `i18n.t()`.
- Файлы локализации дополнены ключами для статус-бара (как на английском, так и на русском).
- При запуске плагина статус-бар отображает корректные сообщения на выбранном языке (по умолчанию английский).
- При смене языка (временно через `i18n.setLocale('ru')`) статус-бар переключается на русский.
- Контекстное меню работает и отображает переведённые пункты.

### Проверка
- Открыть любую заметку без FSRS-полей – статус-бар должен показывать "FSRS: not FSRS", тултип – "FSRS plugin - current file is not a FSRS card".
- Открыть заметку с FSRS-карточкой, которая готова к повторению – "FSRS: due!", тултип – "FSRS plugin - card is due...".
- Повторить карточку – статус-бар должен показывать дату следующего повторения (например, "FSRS: 2025-04-22").
- Если настроен минимальный интервал досрочного повторения (например, 40 минут) и прошло меньше, статус-бар должен показывать "FSRS: Wait 35 minutes".
- Правый клик по статус-бару – меню с пунктами на выбранном языке.
- Временно изменить локаль на 'ru' и убедиться, что все сообщения стали русскими.
